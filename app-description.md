# iOSCryptoApp

Secure encryption and digital signature app for iOS.

## Features

- **AES Encryption**: Encrypt and decrypt text with AES-256
- **Digital Signatures**: Create and verify ECDSA signatures
- **File Encryption**: Encrypt and decrypt files securely
- **Face ID / Touch ID**: Protect access to your keys
- **Secure Storage**: Keys stored in iOS Keychain
