# JavaScript Mastery

Skill completa per padroneggiare JavaScript, dal base al moderno ES6+.

## Contenuti

### Fondamenti
- Sintassi base e variabili (var, let, const)
- Tipi di dati primitivi
- Operatori e control flow
- Funzioni (function declaration, expression, arrow)

### Intermedio
- Array e metodi (map, filter, reduce)
- Oggetti e prototype
- DOM manipulation
- Event handling

### Moderno (ES6+)
- Arrow functions
- Destructuring
- Template literals
- Async/await e Promises
- Modules (import/export)
- Optional chaining e nullish coalescing

### Frontend
- React basics
- State management
- Hooks (useState, useEffect)
- Componenti e props

### Backend
- Node.js fundamentals
- Express.js
- REST API design
