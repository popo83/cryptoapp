# Python Mastery

Skill completa per padroneggiare Python, dal base al avanzato.

## Contenuti

### Fondamenti
- Sintassi base e variabili
- Tipi di dati (stringhe, liste, dizionari, tuple)
- Control flow (if, for, while)
- Funzioni e parametri

### Intermedio
- Programmazione orientata agli oggetti (OOP)
- Gestione errori e eccezioni
- Moduli e pacchetti (pip, venv)
- File I/O

### Avanzato
- Decoratori e generatori
- Context managers
- Concurrency (threading, asyncio)
- Type hinting

### Data Science & ML
- NumPy e Pandas
- Matplotlib per grafici
- Introduzione al Machine Learning
