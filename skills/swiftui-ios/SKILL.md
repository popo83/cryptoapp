# SwiftUI for iOS

Skill per imparare SwiftUI, il moderno framework UI di Apple per creare interfacce iOS.

## Contenuti principali

- View e modificatori
- State e Binding
- Gestione dati con @StateObject
- Navigazione e routing
- Animazioni
- Integrazione con UIKit
